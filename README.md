# Kiro Flappy Game 🤖

**🚀 このゲームはKiro CLI（AWS生成AI）によって完全に生成されました！**

シンプルなFlappy Birdスタイルのブラウザゲームです。人工知能が数分でコーディングしました。

https://main.d2bsfw474omsfw.amplifyapp.com/

## ✨ AI生成の特徴

- **完全自動生成**: Kiro CLIがゲームロジックからHTML/CSSまで全て作成
- **ゼロからの実装**: 人間のコーディング作業なし
- **即座にプレイ可能**: AI生成後すぐに動作するゲーム

## 🎮 ゲーム機能

- HTML5 Canvasを使用したスムーズな描画
- キーボード操作（スペースキーでジャンプ）
- リアルタイムスコア表示
- ゲームオーバー・リスタート機能

## 🕹️ プレイ方法

1. `index.html`をブラウザで開く
2. スペースキーを押して鳥をジャンプさせる
3. パイプに当たらないように進んでハイスコアを目指そう！

## 📁 ファイル構成

- `index.html` - メインのHTMLファイル（AI生成）
- `game.js` - ゲームロジック（AI生成）
- `package.json` - プロジェクト設定
- `amplify.yml` - AWS Amplify設定

## 🚀 デプロイ
